#  EcoVida - Proyecto de Práctica Web

##  Descripción

EcoVida es una página web informativa desarrollada como parte de un proceso de aprendizaje en desarrollo web. El proyecto simula una empresa enfocada en sostenibilidad ambiental y permite aplicar conceptos básicos de maquetación y estilos.

---

##  Objetivo del proyecto

Este proyecto tiene como finalidad reforzar habilidades en:

* Estructura de páginas con HTML
* Estilización con CSS
* Uso de Flexbox para distribución de elementos
* Organización visual de contenido
* Buenas prácticas básicas en diseño web

---

##  Tecnologías utilizadas

* HTML5
* CSS3

---

##  Estructura del proyecto

```
EcoVida/
│
├── index.html
├── css/
│   └── style.css
├── assets/
│   ├── img/
│   └── fonts/
└── paginas/
    └── equipo.html
```

---

##  Secciones implementadas

* **Header:** Logo, nombre y navegación
* **Banner:** Presentación de la misión
* **Servicios:** Tarjetas informativas
* **Productos:** Productos destacados
* **Clientes:** Testimonios
* **Footer:** Información de contacto

---

##  Características del desarrollo

* Uso de Flexbox para organización del layout
* Aplicación de estilos personalizados
* Efectos hover en botones y tarjetas
* Navegación interna mediante anclas (#)
* Implementación de tipografías externas

---

##  Enfoque de aprendizaje

Este proyecto forma parte de un proceso formativo, por lo que:

* No es completamente responsive
* Se prioriza la práctica de conceptos básicos
* Se enfoca en la comprensión del layout y estilos
* Sirve como base para futuros proyectos más avanzados

---

## 🚀 Ejecución

1. Descargar o clonar el proyecto
2. Abrir el archivo `index.html` en el navegador
